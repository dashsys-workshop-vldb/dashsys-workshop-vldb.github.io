# DASHSys 2026 — Dataset

This directory contains the dataset for the DASHSys 2026 Systems Track.

- `data.json` — Illustration examples with full agent trajectories (training set)
- `test.json` — Test set queries
- `DBSnapshot/` — Knowledge Graph snapshot (Parquet files)
